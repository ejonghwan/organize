1) 기본 체크박스

*키보드 사용

Tab : 다음 체크박스로 초점 이동

Space bar : 체크박스 활성화

Tabindex="0" : 초점을 받지 않는 요소에 적용 시 키보드 접근 가능

*구성 요소
기본 체크박스 wai-aria 구성요소 필수유무, role, property, state, 설명 정보 제공
필수유무 role property state 설명
role="group" 동일 속성의 집합
필수 role="checkbox" 체크박스 역할
필수 aria-checked
  • true : 체크됨
  • false : 체크되지 않음
aria-labelledby
또는
aria-label		 role=“group” 사용시 레이블 제공
role="none" ul > li 구조에서 해당 역할을 하지 않는 경우 필수
관심 서비스
  • 날씨
  • 영화
  • 밴드
  • 오피스
